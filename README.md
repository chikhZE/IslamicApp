# Al-Jami' Al-Islami (الجامع الإسلامي) 🌙

A beautiful, modern, and minimalist Islamic Android application engineered with an offline-first architecture. The app provides a distraction-free, calm environment for daily Dhikr, Hadith exploration, and Quranic supplications, respecting the user's focus and device resources.

##  Features

- **🧠 Smart Home Dashboard:** Displays a dynamic Hijri date, total read Adkar counter, and a curated contextual "Hadith of the Day", "Ayah of the Day", and "Dua of the Day".
- **📚 Integrated Forty Hadiths (الأربعين النووية):** A structured, easy-to-read directory of Hadiths accompanied by an in-depth explanation (شرح الحديث) for every text.
- **📿 Legal Adkar & Supplications (الأذكار الشرعية):** Categorized sections for Morning/Evening Adkar, Post-Prayer, Tasbih, Sleep, Awakening, and Quranic/Prophetic supplications.
- **🔄 Interactive Counters:** Real-time progress counters built into individual Adkar cards with a dedicated option to easily share texts.
- **⚙️ Dynamic Theme Engine:** Full support for System Settings, Dedicated Dark Mode, and Light Mode to maximize visual comfort during night reading.
- **🌍 Full RTL Support:** Designed from the ground up natively supporting Right-to-Left layouts for optimal Arabic typography and accessibility.
